O objetivo desta API é, por meio de jobs agendados pelo node-cron, consultar o banco de dados e salvar os respectivos dados consultados no SCPC Serasa / Boa Vista para faturas de crédito / débito.

# Instruções referente à API

### 🔌 Preparando o servidor

**Pré-requisitos:**
* NodeJS >= 12.18.3
* Yarn >= 1.22.4
* Docker >= 19.03.8

### 🔐 Rodando a aplicação

```bash
# Clone o projeto
git clone https://git-codecommit.us-east-1.amazonaws.com/v1/repos/telecoms-schedule-api

# Entre na pasta
$ cd telecoms-schedule-api

# Instale as dependências
$ yarn # ou opte pelo npm install

# Rode os contêiners
$ sudo docker-compose up -d
```

## Comando do Servidor

Quando tudo estiver finalizado, inicie o servidor
```bash
$ yarn dev
```

## Comando de Build

Para buildar a aplicação, rode o comando abaixo:

```bash
$ yarn build
```

Uma pasta **dist** será gerada com o conteúdo TypeScript transpilado.
