## ✅ Avaliação

- Dificuldade de implementação: **Fácil**
- Dificuldade de manutenção: **Fácil**:
- Tempo médio de resposta da API: **208ms**
- Linguagem da API: **JSON ou XML**

## Exemplo de requisição:

**[GET]** https://service.zipcode.com.br/RestService.svc/ConsultaPF/json

### Query Params:

```shell
?token=<token> # Token fornecido pela empresa
?cpf=<cpf> # CPF a ser pesquisado
```

## JSON Entrada:

Exemplo utilizando `axios`

```js
class GetDataTransUnionService {
  async get(request: Request, response: Response) {
    const { cpf } = request.params;

    const transUnionResponse = await axios.get(
      String(process.env.TRANS_UNION_URL),
      {
        params: {
          token: process.env.TRANS_UNION_TOKEN,
          cpf,
        },
      },
    );

    return response.json(transUnionResponse.data);
  }
}
```
